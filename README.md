# IBVS Air Hockey

Image-Based Visual Servoing (IBVS) implementation for air hockey robot control.

## Description

This repository contains the implementation of IBVS control for an air hockey playing robot.

## Files

- `env_creation.py`: Environment setup and configuration
- `Assignment 3 Visual Servoing Problem Set.ipynb`: Jupyter notebook with problem set solutions

## Setup

Install required dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the environment creation script:
```bash
python env_creation.py
```
